# Streamlit_PROJECT
Pour le projet la chose la plus dur à été de travailler avec l'enorme dataset , donc pour le projet j'ai pris un sample de 700000 lignes aléatoire que j'ai ensuite
sort et group par code département.
Je me suis ensuite débarassé des collones que j'ai juger inutile et j'ai créer mes propres collones m2 ...
j'ai ensuite créer différentes fonctions permmettant de créer des maps, des graphiques (bar chart, histogram, line chart ...) grâce à la doc streamlit que j'ai mis en dehors du main.
Ceci m'a permis d'avoir un temps de log court et un chargement très rapide, j'ai ensuite étudié la dataset et j'ai voulu créer un site comme un guide des informations à savoir sur la dataset.
J'avais pour but de faire comprendre la dataset à travers le site sans l'avoir vu préalable.
Mon site est donc une réunion des informations à connaitre sur la dataset tel que le classement des ventes par ville.
Un autre problème à été de mettre les données sur github et streamlit, pour ce faire j'ai du sampler le scv jusqu'à ce qu'on puisse l'utiliser et l'afficher sur le site.
